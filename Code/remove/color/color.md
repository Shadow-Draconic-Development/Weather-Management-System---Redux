<h1>Color Subalias<img align="right" src="../../../Data/images/main.png" width="100px"></h1>

Removes the color hexcode from the specified season (requires Dragonspeaker)

## Usage
`!weather remove color [season name] [color hex]`

- `season name`
    - Required
    - Name of season
- `color hex`
    - Required
    - Hex Code
