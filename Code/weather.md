<h1>Weather Alias<img align="right" src="../Data/images/main.png" width="100px"></h1>

Main content library alias for generating weather reports

## Usage
`!weather [season name] <#>`
- `season name`
    - Required
    - Name of season
- `#`
    - Optional
    - Page number (multiple pages can be generated)
